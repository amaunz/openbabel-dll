Please use the github 'download' button (marked with a green arrow) instead of downloading single files.
IMPORTANT! After downloading, check the file integrity:
- Get ftp://ftp.gnupg.org/gcrypt/binary/sha1sum.exe 
- Run 'sha1sum openbabel-dll.zip'
- Compare output to shasum.txt. If equal, openbabel-dll.zip was received ok.
